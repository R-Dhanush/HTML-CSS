Paragraph tag

The "p" tag is used to display para of text.

For example:paragraph consists of one or more sentences that form a self-contained unit of discourse. The start of a paragraph is indicated by a new line.

Text is easier to understand when it is split up into units of text. For example, a book may have chapters. Chapters can have subheadings. Under each heading there will be one or more paragraphs.

Bold tag

By enclosing words b/w b tags we can make words appear BOLD.

Italic tag

By enclosing words b/w i tags we can make words appear italic.

Superscript tag

By enclosing characters b/w sup we can make characters that should be superscript.For example: 30th July 2004.

Subscript tag

By enclosing characters b/w sub we can make characters that should be subscript.For example: 6:30am.

Line break tag

The br tag is used to add line break in middle of a para.
This elements are known as empty elements.
Because this tags have only opening tag.

Horizontal tag

The hr tag is used to add a horizontal line b/w sections.

There are some text elements that will not affect the structure for the page but it will add extra information to the page.this elements are known as Semantic elements.

Strong tag

The strong tag is used to give a strong importance to a word.

Emphasized tag

The em tag is used to define emphasized text.

Quote tag

The q tag is used to define shorter quotes.For example:Oscar Wilde said Be yourself; everyone else is already taken.

Blockquote tag

The blockquote tag is used to define longer quotes.For example:Marilyn Monroe said

“I'm selfish, impatient and a little insecure. I make mistakes, I am out of control and at times hard to handle. But if you can't handle me at my worst, then you sure as hell don't deserve me at my best.”

Abbreviation tag

The abbr tag is used for abbreviation.For example:Srinivasa Ramanujan 1887 - 1920.A largely self-taught pure mathematician, he enriched number theory with thousands of new identities, equations and theorems.

The abbr tag is also used for acronym words .For example:URL.

Cite tag

The cite tag is used for title of a work.For eample:Badass[from Leo] By Anirudh Ravichander.

Definition tag

The dfn tag is used to define instance of a new term.For example:A black hole is a region of space where the gravitational pull is so strong that nothing, including light, can escape it. It is a cosmic body of extremely intense gravity from which nothing, including light, can escape. Black holes can be observed by the effects of their enormous gravitational fields on nearby matter.
